# API Documentation

Folder ini berisi semua file PHP untuk backend:
- connection-pdo.php → koneksi database
- login.php → API login user
- menu.php → API daftar menu
- order.php → API untuk order

## Cara Menjalankan
1. Buat Folder untuk api ini di C:/xampp/htdocs/
1. Copy folder ini ke: C:/xampp/htdocs/nama_folder
2. Jalankan XAMPP (Apache + MySQL)
3. Akses API:
   http://localhost/nama_folder/login.php

   localhost tergantung di run dimana, misal kalau pake usb debug kehp pake ipv4,harus sama dengan yang dilaptop
